# Práctica 1: Creación de Clases y Encapsulamiento en PHP

## Objetivo

Aplicar los fundamentos de Programación Orientada a Objetos en PHP mediante la creación de una clase que implemente encapsulamiento, atributos privados, constructor y métodos de acceso (getters y setters).

## Descripción de la clase

Se creó una clase llamada **Usuario** que contiene los atributos:

- nombre
- correo

Los atributos se declararon como **privados** para aplicar encapsulamiento.

La clase incluye: 


- Constructor para inicializar los datos
- Métodos **getters** para obtener valores
- Métodos **setters** para modificar valores
