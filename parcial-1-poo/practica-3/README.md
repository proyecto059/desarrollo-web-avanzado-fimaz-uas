# Práctica 3: Sistema de Usuarios con Validaciones y Excepciones

## Descripción

Se desarrolló un sistema orientado a objetos en PHP que implementa herencia, validaciones y manejo de excepciones.

El sistema incluye:

- Clase base `Usuario`
- Clases derivadas `Admin` y `Alumno`
- Validación de correos electrónicos
- Manejo de errores mediante `try/catch`

## Manejo de errores

Se utilizan bloques `try/catch` para capturar excepciones cuando se intenta crear un usuario con datos inválidos.
